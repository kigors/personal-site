Description:

Model is based on trained VGGFace model using transfer learning technique by adding and retraining 2 extra fully connected layers with 3 outputs for:
  • gender prediction (binary classifier)
  • race prediction (categorical classifier)
  • age prediction (regression)
All 3 outputs are trained simultaneously. The latest accuracies were 94% for gender, 86% race and 0.08 mae (on 0-1 scaled age).
